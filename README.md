# Шаблонный сборка на Webpack 5

Шаблонная сборкв на webpack 5 для создания фронтенда сайта с использованием:
- шаблонизатора Pug 
- препроцессора SCSS
- языка разметки Markdown 
- языка JavaScript 
- настройка для React 
- оптимизация изображений 
- подключение шрифтов 
- настройки локального сервера
