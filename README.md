# Шаблонная сборка на Webpack 5

Шаблонная сборка на webpack 5 для создания фронтенда сайта использует следующие компоненты:
- шаблонизатор Pug 
- препроцессор SCSS
- язык разметки Markdown 
- язык JavaScript 
- настройка для React 
- оптимизация изображений 
- подключение шрифтов 
- настройка локального сервера
